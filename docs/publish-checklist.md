# 公開前チェックリスト

公開前に確認する項目をまとめます。

## サイト設定

- `content/data/site.json` の `name` が確定している
- `content/data/site.json` の `shortName` がヘッダー表示に合っている
- 公開ドメインが決まったら `url` を設定する
- OGP画像の文言がサイト名とずれていない

## 法務・信頼性

- 免責事項ページが表示される
- 広告・PRポリシーページが表示される
- プライバシーポリシーページが表示される
- 記事詳細に免責文が表示される
- 広告やPRを入れる場合は、記事内で明示する

## コンテンツ

- 初めて読む人向けの記事がある
- 運用方針ページに年間予算、銘柄配分、やらないことがある
- プロフィールで始めた理由が説明されている
- 月次ログの記事テンプレートが使える
- 下書き記事は `isPublished: false` にする

## 表示確認

- トップページがPCで崩れていない
- トップページがスマホで横スクロールしない
- 記事詳細が読みやすい
- 運用ログページが表示される
- 表がスマホで横スクロールできる
- 404ページが表示される

## 生成・公開

- `node scripts/build.mjs` が成功する
- `SITE_URL=https://example.com node scripts/build.mjs` で `sitemap.xml` と `feed.xml` が生成される
- ローカルリンク切れがない
- 公開先に `dist/` の中身を配置する
